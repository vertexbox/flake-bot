# compliance-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) serves to audit and regulate compliance status for repos in [daeuniverse](https://github.com/daeuniverse) organization.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm run debug
```
